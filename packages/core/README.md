# @windease/core

Headless, framework-agnostic core for [windease](https://github.com/orochi235/windease).

```sh
npm install @windease/core
```

Ships the node store, layout-strategy contract, snapshot v2 (with v1
migration), and the built-in strategies (`gridStrategy`, `stackStrategy`,
`stripStrategy`, `binarySplit`, `recursiveSplit`).

No React, no DOM. For React bindings see
[`@windease/react`](https://www.npmjs.com/package/@windease/react).

Docs and a live playground: <https://github.com/orochi235/windease#readme>.
