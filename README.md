# windease

Browser-based window manager. Framework-agnostic core (`@windease/core`) plus a
React binding (`@windease/react`). Zones are first-class containers; each
window carries small per-concern FSMs (lifecycle, transit, focus).

See `docs/superpowers/specs/2026-06-04-windease-design.md` for the design.

## Develop

```bash
npm install
npm test
npm run build
```
